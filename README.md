Displays Now Showing, Popular and Upcoming movies
## 3rd party libraries used:
* Retrofit - Networking
* Glide - Images
## API:
Used The Movies Data Base(TMDB) API(https://www.themoviedb.org/documentation/api) to get all the data required for this application
